# Cyber Security Tool

간단한 보안 점검과 자가진단을 위한 **Python 기반 사이버 시큐리티 툴**

## 프로젝트 소개
**Cyber Security Tool**은 사용자가 자신의 시스템과 웹 환경을 직접 점검하고, 기본적인 보안 상태를 스스로 확인할 수 있도록 제작한 프로그램입니다.

최근 해킹과 정보 유출 사고가 자주 발생하고 있지만, 일반 사용자가 전문적인 보안 도구를 직접 다루기에는 어려움이 있다고 생각했습니다. 그래서 복잡한 보안 솔루션이 아니더라도, 사용자가 자신의 환경에서 위험 요소를 직접 확인하고 간단한 자가진단을 할 수 있는 도구를 만들어보고자 했습니다.

또한 정보보안 분야를 공부하는 과정에서 배운 내용을 실제 기능으로 구현해보며 실습 경험을 쌓고, 이를 포트폴리오로 활용하고자 이 프로젝트를 진행했습니다.

## 제작 목적
- 사용자가 자신의 시스템 환경을 직접 점검할 수 있는 보안 도구를 구현해보기 위해 제작했습니다.
- 정보보안 분야에 대한 관심을 바탕으로, 공부한 개념을 실제 프로그램으로 구현하며 실습 경험을 쌓고자 했습니다.
- 포트 점검, 위험도 분석, 보고서 저장 등의 기능을 직접 개발하면서 보안 점검 방식과 위험 요소를 더 깊이 이해하고자 했습니다.

## 주요 기능
- 포트 스캔
- 위험 포트 탐지
- 리스크 스코어 표시
- 로그 출력
- 보고서 저장
- 웹 스캔
- 비밀번호 보안 수준 평가
- 해시 생성

## 기능 설명

### 1. 포트 스캔
사용자가 입력한 IP 주소와 포트 범위를 기준으로 열려 있는 포트를 확인할 수 있도록 구현했습니다.  
이를 통해 시스템에서 외부와 연결 가능한 포트를 점검할 수 있습니다.

### 2. 위험 포트 탐지
보안상 주의가 필요한 포트를 탐지하여 사용자에게 표시하도록 구성했습니다.  
단순히 열린 포트를 확인하는 것에서 끝나는 것이 아니라, 주의가 필요한 포트를 빠르게 파악할 수 있도록 했습니다.

### 3. 리스크 스코어 표시
탐지된 포트와 조건에 따라 점수를 부여하고, 이를 기반으로 전체 위험도를 계산하도록 구현했습니다.  
사용자는 수치화된 결과를 통해 보안 상태를 보다 직관적으로 이해할 수 있습니다.

### 4. 로그 출력 및 보고서 저장
스캔 과정과 결과를 로그 형태로 확인할 수 있도록 했으며, 결과를 HTML 보고서 형식으로 저장할 수 있도록 구성했습니다.  
이를 통해 점검 기록을 남기고, 이후 다시 확인할 수 있습니다.

### 5. 웹 스캔
웹 환경에서 확인 가능한 기본적인 보안 요소를 점검할 수 있도록 기능을 추가했습니다.  
이를 통해 네트워크 점검뿐 아니라 웹 관련 보안 요소도 함께 확인할 수 있도록 했습니다.

### 6. 비밀번호 보안 수준 평가
사용자가 입력한 비밀번호의 길이와 문자 조합을 기준으로 보안 수준을 확인할 수 있도록 구현했습니다.  
안전한 비밀번호의 중요성을 직접 확인할 수 있도록 돕는 기능입니다.

### 7. 해시 생성
입력한 문자열을 해시 값으로 변환할 수 있도록 구현했습니다.  
이를 통해 해시의 개념을 직접 확인하고, 정보보안 분야에서 사용되는 기초 기능을 경험할 수 있도록 했습니다.

## 구현 방법
- `Python`을 사용하여 전체 프로그램을 개발했습니다.
- `socket` 라이브러리를 활용하여 포트 스캔 기능을 구현했습니다.
- `tkinter`를 이용해 GUI 환경을 구성했습니다.
- 특정 포트 번호와 조건에 따라 위험 점수를 부여해 리스크 스코어를 계산하도록 했습니다.
- 스캔 결과를 로그로 출력하고, 보고서 형태로 저장할 수 있도록 구현했습니다.
- `requests`를 활용해 웹 관련 기본 점검 기능을 추가했습니다.
- 비밀번호 보안 수준 평가 기능과 해시 생성 기능을 함께 구성했습니다.

## 사용 기술
- Python
- tkinter
- socket
- requests
- hashlib

## 기대 효과
- 사용자가 자신의 환경을 직접 점검하며 기본적인 보안 상태를 확인할 수 있습니다.
- 위험 포트와 위험도를 직관적으로 파악할 수 있습니다.
- 결과를 저장하고 확인하면서 점검 기록을 관리할 수 있습니다.
- 정보보안 기초 개념을 실제 프로그램으로 구현하며 학습 효과를 높일 수 있습니다.
