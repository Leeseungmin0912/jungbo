# Syber Security Tool

## 프로젝트 소개
정보보안에 관심을 가지면서, 사용자가 스스로 자신의 환경을 간단하게 점검해볼 수 있는 사이버 시큐리티 툴을 제작했습니다.  
최근 해킹 사례가 많아지는 상황에서, 복잡한 보안 도구가 아니더라도 기본적인 위험 요소를 직접 확인하고 자가진단할 수 있는 프로그램이 있으면 좋겠다고 생각해 만들었습니다.

또한 정보보안 분야를 공부하는 과정에서, 제가 배운 내용을 직접 구현해보며 포트폴리오로도 활용하고자 이 프로젝트를 진행했습니다.

## 이 툴을 만든 이유
1. 최근 해킹 피해 사례가 많아지면서, 사용자가 스스로 보안 상태를 간단히 점검할 수 있는 프로그램을 만들어보고 싶었습니다.
2. 정보보안 분야에 대한 관심을 바탕으로 공부한 내용을 직접 구현해보며, 기초적인 보안 포트폴리오를 만들고 싶었습니다.
3. 아직 전문적인 지식이 많지 않더라도, 직접 도구를 제작하면서 보안 점검 방식과 위험 요소를 더 잘 이해할 수 있다고 생각했습니다.

## 주요 기능
- 포트 스캔
- 위험 포트 탐지
- 리스크 스코어 표시
- 로그 출력
- 보고서 저장

## 구현 방법
- `Python`을 사용하여 프로그램을 개발했습니다.
- `socket` 라이브러리로 포트 스캔 기능을 구현했습니다.
- `tkinter`를 이용해 GUI 환경을 구성했습니다.
- 특정 포트 번호와 조건에 따라 위험 점수를 부여해 리스크 스코어를 계산하도록 했습니다.
- 스캔 결과를 로그로 출력하고, 결과를 `txt` 또는 `html` 형식의 보고서로 저장할 수 있도록 구현했습니다.

## 프로젝트를 통해 배운 점
이 프로젝트를 통해 포트 스캔의 기본 원리와 네트워크 점검 방식에 대해 더 이해할 수 있었고,  
단순히 이론으로 배우는 것보다 직접 구현해보는 과정에서 정보보안에 대한 흥미와 이해도가 더 높아졌습니다.
