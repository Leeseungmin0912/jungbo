# Cyber Security Tool
-# 간단한 보안 점검과 자가진단을 위한 Python 기반 사이버 시큐리티 툴

## 프로젝트 소개
정보보안에 관심을 가지면서, 사용자가 자신의 환경을 간단하게 점검할 수 있는 사이버 시큐리티 툴을 제작했습니다.  
최근 해킹 피해 사례가 많아지는 상황에서, 복잡한 보안 도구가 아니더라도 기본적인 위험 요소를 직접 확인하고 자가진단할 수 있는 프로그램이 있으면 좋겠다고 생각해 만들었습니다.

또한 정보보안 분야를 공부하는 과정에서, 배운 내용을 직접 구현해보며 포트폴리오로도 활용하고자 이 프로젝트를 진행했습니다.

## 제작 이유
1. 최근 해킹 피해 사례가 많아지면서, 사용자가 스스로 보안 상태를 간단히 점검할 수 있는 프로그램을 만들어보고 싶었습니다.
2. 정보보안 분야에 대한 관심을 바탕으로 공부한 내용을 직접 구현해보며, 기초적인 보안 포트폴리오를 만들고 싶었습니다.
3. 직접 도구를 제작하는 과정에서 보안 점검 방식과 위험 요소를 더 깊이 이해할 수 있다고 생각했습니다.

## 주요 기능
- 포트 스캔
- 위험 포트 탐지
- 리스크 스코어 표시
- 로그 출력
- 보고서 저장
- 웹 스캔
- 비밀번호 보안 수준 평가기
- 해시 생성기

## 구현 방법
- `Python`을 사용하여 프로그램을 개발했습니다.
- `socket` 라이브러리를 활용해 포트 스캔 기능을 구현했습니다.
- `tkinter`를 이용해 GUI 환경을 구성했습니다.
- 특정 포트 번호와 조건에 따라 위험 점수를 부여해 리스크 스코어를 계산하도록 했습니다.
- 스캔 결과를 로그로 출력하고, 결과를 `txt` 또는 `html` 형식의 보고서로 저장할 수 있도록 구현했습니다.
- 웹 스캔 기능을 통해 웹 환경에서 확인 가능한 기본적인 보안 요소를 점검할 수 있도록 구성했습니다.
- 비밀번호 보안 수준 평가기를 통해 사용자가 입력한 비밀번호의 강도를 확인할 수 있도록 했습니다.
- 해시 생성 기능을 추가해 문자열을 다양한 해시 형식으로 변환할 수 있도록 구현했습니다.

## 프로젝트를 통해 배운 점
이 프로젝트를 통해 포트 스캔의 기본 원리와 네트워크 점검 방식에 대해 더 깊이 이해할 수 있었습니다.  
또한 단순히 이론으로 배우는 것보다 직접 구현해보는 과정에서 정보보안에 대한 흥미와 이해도가 더욱 높아졌습니다.

무엇보다 정보보안은 단순히 재미로 접근할 분야가 아니라, 사용자의 정보를 안전하게 보호해야 하는 책임이 따르는 분야라는 점을 다시 느끼게 되었습니다.  
이 프로젝트를 진행하면서 개발자는 기능을 만드는 것에 그치는 것이 아니라, 사람들의 정보를 지키기 위한 책임감 있는 자세로 프로그램을 만들어야 한다는 점을 배우게 되었습니다.  
이를 통해 앞으로도 정보보안 분야를 더 진지한 태도로 공부하고, 신중하게 개발해야겠다는 생각을 가지게 되었습니다.
